# Mobile Phones Retail Sales Performance Dashboard

## Project Overview
This Power BI dashboard provides an executive-level analysis of mobile phone sales across major global brands (Apple, Samsung, OnePlus, Vivo, and Xiaomi). It tracks gross revenue, transaction volumes, and regional distribution across multiple cities to identify top-performing brands, popular mobile models, and purchasing trends.

## Files in this Folder
* `Mobile_Sales_Dashboard.pbix` - The core interactive Power BI application containing the finalized data model and visualization sheets.
* `Mobile Sales Data.xlsx` - The underlying mobile retail sales dataset used for building the data model.

## Skills & Technical Competencies Demonstrated
* **Multivariate Data Analysis:** Analyzing sales performance across competing product brands and individual device models simultaneously.
* **Geographical Data Mapping:** Utilizing built-in map visualizations to display sales concentrations by major metropolitan hubs (Delhi, Mumbai, Bangalore, Chennai, etc.).
* **Advanced Filtering & Slicers:** Implementing month-of-year and payment type (Cash, Credit Card, Debit Card, UPI) filters for granular data exploration.
